# 闻一翻译：项目介绍

闻一翻译是一个面向学术文献的翻译工作台。

当前版本已经具备一条完整的基础链路：用户可以上传 `PDF / DOCX / MD / TXT` 学术文档，系统会把文档解析为 Markdown，再进一步拆成结构化 Block，完成文献总结、翻译提示词生成、逐块翻译、对照阅读、人工编辑和 PDF 导出。它更适合拿来做“先得到一版可阅读、可核对、可继续修改的译稿”，而不是把翻译结果一次性定稿。

但我们也不想把已经变成行业默认项的能力误写成亮点。仅仅是“支持 PDF 翻译、保留基础版式、左右对照阅读、文献总结”，已经很难构成真正的竞争力。因此，闻一翻译接下来更想做深的，不是“再做一个通用翻译网站”，而是继续沿着以下方向演进：

- 长文献 / 外文专著翻译工作台
- 更适合严肃校订的可信翻译系统
- 术语一致性与风险标注能力
- 面向高校、实验室、出版机构的私有部署与协作工作流

如果把这个项目放到更清晰的位置上，我们更愿意这样介绍它：

> 闻一翻译不是“翻译一篇论文”的工具，而是“处理一整本文献翻译工程”的工作台雏形。
