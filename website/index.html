Academic Translation Workspace

先读懂论文，再逐块翻译。

闻一翻译将 PDF / DOCX 学术文献转换为 Markdown，再拆成可对应的 Block。系统会先总结文档内容，识别领域和作者写作特点，构建专属翻译提示词，然后逐块翻译，最终形成左原文、右译文的一一对应阅读体验。

核心流程

  1. 上传 PDF / DOCX / Markdown 文档
  2. 解析为 Markdown 并构建 Block 树
  3. 生成文献总结和翻译提示词
  4. 逐块翻译并保存进度
  5. 查看左右对照翻译结果
